screen :
	screen -S name				新建session
	screen -r name				进入session
	screen -ls					session列表
	screen -X -S name quit		删除session
	ctrl + a + d				退出当前session
	screen -D -r name			进入上次未退出session

	
用 screen -ls, 显式当前状态为Attached，但当前没有用户登陆些会话。screen此时正常状态应该为(Detached)此时用screen -r <session-id>，怎么也登不上。最后找到解决方法： screen -D  -r ＜session-id>

-D -r  先踢掉前一用户，再登陆。
